## Development and testing

#### 1. Clone the repository

```
git clone git@github.com:jpwilliams/fuzzy-tabber.git
cd fuzzy-tabber
```

#### 2. Install dependencies

```
yarn
```

#### 3. Build

```
yarn build
```
